A secure and customizable password generator built with HTML, CSS, and JavaScript.

Uppercase letters
Lowercase letters
Numbers
Special characters

Copy password to clipboard with one click
Visual password strength indicator
Mobile-responsive design

CSS3
JavaScript (ES6+)
